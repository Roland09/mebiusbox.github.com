# UML

`uml` プラグインを使います．

```json
{
    "plugins": "uml"
}
```

次のように使います．

{% raw %}
```uml
@startuml

	Class Stage
	Class Timeout {
		+constructor:function(cfg)
		+timeout:function(ctx)
		+overdue:function(ctx)
		+stage: Stage
	}
 	Stage <|-- Timeout

@enduml
```
{% endraw %}

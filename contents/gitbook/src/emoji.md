# 絵文字

`advanced-emoji` プラグインを使います．

```json
{
    "plugins": ["advanced-emoji"]
}
```

次のように使います．

<!-- ignore:advanced-emoji:start -->
```
:smile:, :heart_eyes:, :blush:
```
